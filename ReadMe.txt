Included:
- OS.java: Driver program of the application.
- Scheduler.java: Scheduler class: Includes six scheduling classes.
- Job.java: Job class: Contains job information.
- Project 3 Summary: Summary of the project.

Follow these steps to run the application:
1. Place following files in the same folder:
	- OS.java
	- Scheduler.java
	- Job.java
2. Run following commands:
	- javac OS.java Scheduler.java Job.java
	- java OS
